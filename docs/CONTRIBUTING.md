# Contributing to Zip Zap Zop

## Naming Variables

|        Thing | Name       |
| -----------: | :--------- |
|     Password | `password` |
|     Username | `username` |
|    User UUID | `uuid`     |
|    Chat UUID | `cuid`     |
| Message UUID | `muid`     |

## Parameters

When passed to a function, `password` should always be encrypted.
